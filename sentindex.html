API Overview

The SentimentPro API provides programmatic access to our sentiment analysis capabilities. Our API is RESTful, uses JSON for requests and responses, and follows standard HTTP response codes.

Base URL
https://api.sentimentpro.com/v1
Response Format

All API responses are in JSON format and include standard response fields:
